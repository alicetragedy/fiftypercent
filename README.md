How to set it up:

`bundle install`

`rake db:create`

`rake db:migrate`

test:

`rake test`

deploy:

`cap production deploy`

deploy branch:

`cap production deploy BRANCH=anja-remove-inuit`

to see the page locally:
`rails s`

to see the 50percent page:

http://localhost:3000

to see the blog

http://blog.local.test:3000/
